- Server setup <br />
npm install body-parser cors express mongoose nodemon bcryptjs jsonwebtoken amqplib <br />
npm start <br />

- Client set up <br />
npm install --legacy-peer-deps <br />
npm start <br />

- Worker setup <br />
npm install nodemon amqplib <br />
npm start <br />

- The message broker is served by RabbitMQ. To install and run a local RabbitMQ server, go to https://www.rabbitmq.com/download.html
